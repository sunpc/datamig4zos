Data initialization

During the first time run, a data folder would be created. The structure can be found as below.

data
+---hsql : HSQL database for Data Migration Tool
|   |   dmigdb.lck
|   |   dmigdb.log
|   |   dmigdb.properties
|   |   dmigdb.script
|   |
|   \---dmigdb.tmp
+---temp : Folder to contain temporary files
\---templates : Folder to contain template files
        template_cntl_allo.jcl
        template_cntl_set.jcl
        template_dataset_ftp.jcl
        template_table_load.jcl
        template_table_query.jcl
        template_table_unload.jcl

Note: The folder and file names can be set up in the Preferences.
