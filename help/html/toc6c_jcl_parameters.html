JCL parameters

Parameters needed on each JCL template:

Common parameters

  • #PROCESS_ID# : process id
  • #JOB_NAME# : job name - the last 6 chars of ProcessIdentifier with a 2-char prefix
  • #BATCH_ID# : job batch id
  • #USER_ID# : user id
  • #CNTL_LIBRARY# : control library
  • #CNTL_MEMBER# : control member name
  • #DB2LOAD# : DB2 load library
  • #DSNLOAD# : DSN load library
  • #SORTLIB# : sort exec library
  • #SPACE# : default space allocation
  • #UNIT# : default DASD unit

    • Unload parameters

  • #DB_SSID# : database SSID
  • #DB_NAME# : database name
  • #DB_SCHEMA# : database schema
  • #TS_NAME# : tablespace name
  • #TABLE_NAME# : table name
  • #DATASET_SYSPUNCH# : sys punch file name
  • #DATASET_SYSREC# : sys rec file name
  • #UNLOAD_SQL# : unload SQL

    • FTP parameters

  • #HOST_NAME# : remote host name
  • #DS_NETRC_NAME# : NETRC dataset name
  • #DS_SOURCE_NAME# : source dataset name
  • #DS_TARGET_NAME# : target dataset name
  • #DS_SOURCE_PREFIX# : source dataset prefix
  • #DS_TARGET_PREFIX# : target dataset prefix
  • #DS_SOURCE_MEMBER# : source dataset member
  • #DS_TARGET_MEMBER# : target dataset member
  • #DS_LRECL# : dataset rec length
  • #DS_BLKSZ# : dataset block size
  • #DS_RECFM# : dataset rec format
  • #DS_SPACE# : dataset space unit
  • #DS_PRI# : dataset primary space
  • #DS_SEC# : dataset secondary space
  • #DS_UNIT# : dataset device unit type
  • #DS_UCOUNT# : dataset device unit count

    • Load parameters

  • #DB_SSID# : database SSID
  • #DB_NAME# : database name
  • #DB_SCHEMA# : database schema
  • #TS_NAME# : tablespace name
  • #TABLE_NAME# : table name
  • #DATASET_SYSPUNCH# : sys punch file name
  • #DATASET_SYSREC# : sys rec file name
  • #DATASET_SYSDISCD# : sys discard file name

    • Query parameters

  • #DB_SSID# : database SSID
  • #DB_SCHEMA# : database schema
  • #DATASET_OUTPUT# : sys print file name
  • #SQL_STMTS# : sql statements
